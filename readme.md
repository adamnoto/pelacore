## Pelacore

Pelacore is taken from an Indonesian word "pelakor" which means an "actor" in English.

Work in Progress

To run the project:

- `yarn run build` at `pelacore-ui`
- `go run ./cmd/app/main.go`

To test:

- `goconvey`
